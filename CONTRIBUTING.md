# Contributing

Thanks for taking a look at LanTransfer.

## Development Setup

```powershell
npm install
npm test
npm start
```

## Useful Commands

```powershell
npm test
npm run build:win
npm run build:site
```

## Pull Request Guidelines

- Keep changes focused.
- Add tests for behavior changes.
- Do not commit generated `dist/`, `node_modules/`, or local output folders.
- Keep UI copy available in both Chinese and English when adding visible text.
- Verify the computer console and phone page at desktop and mobile widths when changing UI.

## Project Principles

- Local-first by default.
- No login requirement.
- No cloud relay in the core transfer path.
- Favor simple browser flows on the phone.
- Keep the Windows package portable.
